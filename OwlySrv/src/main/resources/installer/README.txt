Owly Server Installaton
==============================


1. Copy the OwlySrv_Installer_<version>.tar to the server's folder where we want to run the program.

2. Expand the tar file :
	tar xvf OwlySrv_Installer_<version>.tar

3. Run installer
	- Change binary to 777 --> chmod 777 OwlySrv_Installer_<version>.bin
	- Execute it : ./OwlySrv_Installer_<version>.bin
	
4.- Start the tomcat with the application installed :
	service tomcat7d start
